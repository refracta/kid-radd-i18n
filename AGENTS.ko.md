# Kid Radd 번역 컨벤션 (Korean)

## 유지 원칙
- 번역 작업(신규/수정)을 할 때마다 이 문서를 **지속적으로 업데이트**한다.
- 번역 결정(용어, 어투, 표기)이 바뀌면 먼저 이 문서를 갱신한 뒤 JSON 번역 파일에 반영한다.
- 이 문서를 한국어 번역 품질의 단일 기준(Single Source of Truth)으로 사용한다.

## 등장인물/고유명사 표기
- `Kid Radd` -> `Kid Radd` (작품명은 영문 유지)
- `Radd` -> `래드`
- `Sheena` -> `시나`
- `Gnarl` -> `그날`
- `Gnarlborg 2000` -> `그날보그 2000`
- `player` -> `플레이어`

## 문체 규칙
- 나레이션: 기본적으로 존댓말 서술체(`~습니다/~합니다`)를 사용한다.
- 나레이션: 패널 간 어투를 섞지 않는다(반말/존댓말 혼용 금지).
- 말풍선 대사: 캐릭터 톤을 우선하며 구어체/반말 사용 가능.
- 말풍선 대사: 같은 인물의 말투는 에피소드 내에서 일관되게 유지한다.

## 번역 스타일 가이드
- 직역보다 의미 전달과 리듬을 우선하되, 원문의 개그/톤을 보존한다.
- 과도한 의역, 현대 유행어 남용, 불필요한 장문 설명을 피한다.
- 제목은 짧고 강하게 번역하되, 시리즈 내 톤 일관성을 유지한다.
- 한국어 문장부호/띄어쓰기를 맞추고, 줄바꿈으로 의미가 깨지지 않게 한다.
- 조작/시스템 설명에서 `however`는 단순 역접(`하지만`)보다 문맥에 맞는 대비 표현(`반면`, `대신`)을 우선한다.

## HTML 페이지 번역 규칙
- 비코믹 안내 페이지(`index`, `listp`, `faq`, `credits`, `stuff`, `tweaks`, `making*`)는 `lang/ko/*.htm` 전용 페이지로 관리한다.
- 전용 페이지는 원본 `href` 값을 변경하지 않는다(링크 목적지/경로 유지).
- 전용 페이지에서 상대 경로 해석을 위해 `<base href=\"../../pages/\">`를 사용한다.
- 인라인 강조/서식 태그(`<b>`, `<i>` 등)가 의미를 가지는 문장은 번역에서도 서식을 보존한다.
- `listp.htm`의 연재 기간 표기는 `YYYY년 M월` 또는 `YYYY년 M월~N월` 형식으로 통일한다.

## 적용 체크리스트
- 고유명사 표기가 기존과 일치하는가
- 나레이션이 `~습니다/~합니다`로 통일되었는가
- 말풍선 톤이 캐릭터별로 자연스럽고 일관적인가
- 기존 번역 파일과 용어 충돌이 없는가
- 강조 태그(`<b>`, `<i>`)가 번역 과정에서 사라지지 않았는가
